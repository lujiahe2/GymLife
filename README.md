# GymLife

A web app that helps gym beginners build and follow training plans, with **RAG** (retrieval-augmented generation) for personalized guidance and **gamification** (streaks, progress, unlocks) inspired by apps like Duolingo, in a workout experience similar in spirit to Ladder.

## Goals

- **Structured plans** — workouts and progression tailored to beginners.
- **RAG assistant** — answers and coaching grounded in your knowledge base (exercises, safety, programming).
- **Motivation** — streaks, milestones, and “unlock next level” style progression to keep users consistent.
